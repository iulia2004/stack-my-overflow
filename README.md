# Stack my Overflow

## What it is
Stack my Overflow is a web application where users can ask and answer technology and programming-related questions and also get help from an integrated AI agent

## What it does
Users can post their questions and help others as well. The app includes AI features, like the questions being automatically marked with relevant tags to make browsing easier and an AI user that can suggest answers if it feels confident enough. The users can approve answers if they are satisfied and discuss in the comments

## Tech stack
React, Node.js/Express, Supabase, Groq/Ollama
